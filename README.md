Global Electronics Retailer Dashboard – Design Approach


The objective of this project was to analyze the performance of a global electronics retailer and provide business insights through interactive Tableau dashboards. Before implementation, a low-fidelity wireframe was created using Figma to plan KPI placement, filter positioning, chart selection, and overall dashboard structure.

Dashboard 1 – Profit Analysis Dashboard

This dashboard focuses on Profitability and Revenue Analysis and includes key performance indicators (KPIs) such as Profit, Cost, Revenue, and Profit Margin. To understand the key drivers of business performance, a Treemap (Bubble Chart) is used to visualize Revenue Contribution by Category, a Bar Chart highlights the Top Global Stores by Revenue, a Map analyzes Profit by Country, and a Pie Chart illustrates the contribution of different Genders to Profit and Revenue. Additionally, a Category Performance Summary Table View provides a detailed breakdown of key metrics, including Quantity, Orders, Profit, Revenue, Average Shipment Days, and Average Order Value (AOV) across product categories, enabling deeper comparative analysis and data-driven decision-making.


Dashboard 2 – Sales Performance Dashboard
This dashboard focuses on sales and customer behavior. The KPIs include Total Customers, Average Shipment Days, Total Orders, and Average Order Value (AOV). Line Charts were used to analyze Order Count by Year and Revenue Trends Over Time. A Bar Chart was used to compare Sales by Subcategory, while another Bar Chart was used to identify countries generating the highest order volume.


Filters

Two global filters were implemented:
-Year Filter
-Country Filter
Both filters are synchronized across all visualizations to provide a consistent analytical experience.

Conclusion
This dashboard solution enables stakeholders to monitor profitability, analyze sales performance, identify high-performing regions and categories, and make data-driven business decisions.
